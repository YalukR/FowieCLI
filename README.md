**Fowie** es una aplicación de consola multiplataforma (Linux / Windows) que vigila el rendimiento de tu computadora y te avisa cuando algo anda mal

**Requisitos**

- Python 3.12.3
- pip

Dependencias:

 - ['psutil'](https://pypi.org/project/psutil)

 **Instalación**

 Clona o descarga el proyecto. 
 Entra a la carpeta raíz (donde está "pyproject.toml")

 **Instalar Dependencias**

 pip install psutil

 **Instalar Fowie**

 pip install -e .
